小女皇江映蓉

简介：

09快女终于决出冠亚季军，小女皇江映蓉胜出～经过一个夏天的“厮杀”，不管是冠、亚、季军还是十强的名次都已尘埃落定。哭也罢，笑也好，接下来，她们总归要离开快女这个“生育”她们的舞台，去寻找各自的天空。或许不久的将来她们会成为同门师姐妹，也或者成为竞争对手。

在总决赛之前，21岁的江映蓉也是最被看好的一位，被媒体称为"有望续写李宇春当年神话"。如今大幕落下，江映蓉称自己首先要狠狠睡一觉，”好久没有睡到自然醒了“。

小时逢人就说我给你唱歌呀

如果不是看到江映蓉在送别其他选手时大哭的情景，许多观众都以为江映蓉长了张“笑脸”呢！据湖南卫视工作人员周小姐介绍，不管排练多紧张，舞台上下的江映蓉总是面带笑容，“后来我们听唯唯（江映蓉小名）的妈妈讲，唯唯的性格很像她爸爸，从小就爱笑，嘴巴也很甜，在家附近溜达时，不管认识不认识的，她都要打招呼。有时江妈妈提醒她要矜持些，她才不管呢，而且超级自恋，家里的墙上都是她的艺术照、生活照，小时只要一喊‘唯唯，给你照相’，她马上就跑出来摆各种姿势。”
