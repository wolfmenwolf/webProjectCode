虽然新专辑《张靓颖@音乐》早在上个月已经面试，但上市后还未专门为这张新专辑举行过任何特别的演唱会，今天靓颖弥补上了这一空白。25日晚，在北京恒基商城，张靓颖2009新浪歌会火热唱响，靓颖在现场为大家带来了新专辑中的歌曲，并接受了歌迷们的特别礼物。
今天在现场除了凉粉之外，还有很多人拥在场地外以及恒基商城的2、3层走廊中观看。靓颖带着舞团登场，以一曲《我的音乐让我说》的热舞开场，从一开始就点燃了歌迷热情。这次从歌曲因为已经与大家见面一段时间了，因此歌迷们也非常熟悉，靓颖与台下歌迷有关专辑的对话每次都能得到大家的呼应。
今天的歌会中，歌迷们非常幸运，连续得到和靓颖互动的机会，除了和靓颖互动做游戏之外，他们还带来了几份神秘的礼物：
礼物一：定制CD。新浪乐库提供了在线定制CD的服务，今天两份为靓颖带来了他们的礼物，便是从新浪乐库中选择靓颖的歌曲而特别制作的一张CD，这是在市场上无法买到的绝版。
