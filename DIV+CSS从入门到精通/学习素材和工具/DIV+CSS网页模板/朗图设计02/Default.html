博客朗图 +　博客家园 +　我的博客
　　朗图成立于1999年，成立早期以提供全面性的品牌形象识别设计为工作核心，现已发展成为业务遍及全国各地、拥有80余名创作人员的整合传播机构，通过一体化的系统解决方案，帮助客户建立及传播差异化品牌、创造深度价值！旗下拥有朗图品牌整合推广、柠檬传媒及朗图地产广告三大专业机构。
朗图（Rito）的含义：
R：Runner 领跑者
I：International 国际视野
T：Together 共同成长
O：Open 开放性思维
PARTNERSHIPS 合作伙伴

　　成立10年来，朗图与全球数百家不同领域的优秀企业建立了长期的战略合作伙伴关系，主要客户涵盖地产、商业零售、酒店餐饮、公共事业机构、时装家居、金融、高新科技等数十个行业领域。

　　 经过多年传播实践和前瞻性的业务细分战略，朗图得以在中国创意产业中脱颖而出、稳健前行，体现了强大的平台竞争力：系统服务链+精准策略+出众创意设计+环球资源=朗图竞争力……

　　今年正好是朗图创业的十周年，以我的观点来看，“继往”固然重要，但更为重要的却是“开来”，所以我更愿意思考一下，下一个十年，朗图会怎么样？深圳的设计会怎么样？更大一点的说，中国的设计会怎么样？世界的设计会怎么样？在如今全球化大背景之下，朗图将会发展成一个什么样的企业？对深圳和中国的设计发展我们有一个什么的愿景？我觉得这个目标和规划是不能单单从财务……
