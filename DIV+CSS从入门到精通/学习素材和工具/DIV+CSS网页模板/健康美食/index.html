用微波炉加热鲜奶会致癌？

网络上有人说鲜奶用微波炉加热后喝有致癌的疑虑，是否真的呢？鲜奶以微波加热后饮用是现在常被使用的方法，不过并没有相关文献报导过有致癌的疑虑。相反的，喝牛奶的好处多多，不但富含多种必需营养素和维生素，同时是钙质的良好来源，更有研究指出牛奶能有效预防结肠癌。

　　

　　其实微波加热的原理很简单，乃是利用微波(即所谓的电磁波)引发鲜奶内部水分子的剧烈震荡，水分子间因为震荡而彼此摩擦产生高温(热)，而使鲜奶升温。所以从其加热方式的理论而言，并不会对人体造成所谓致癌性的伤害。

　　

另外，鲜奶即使加热过，影响的只是本身奶蛋白会因为受热而稍稍产生凝结的现象(为蛋白质加热后的凝聚现象，是自然的现象)。通常，要达到蛋白质的凝聚现象要高温加热一阵子，像如果我们以传统的方式用锅子煮牛奶，就会发现加热一段时间后，牛奶表面结了一层奶皮(就像豆浆会煮出豆皮的道理一样)，即使如此，这层奶皮还是可以安全食用的，在体内的消化吸收效果也一样好，所以牛奶用微波加热应可安全饮用无虞。
