<<HTML5+CSS3入门到精通>>光盘内容及其使用方法

光盘内容:

	本套光盘提供了<<HTML5+CSS3从入门到精通>>一书的源程序，包括的目录如下。

	1.本书源代码和示例，提供了本书各章所涉及的实例源程序以及素材文件。
	2.学习素材和工具，提供了网页设计学习素材和参考工具。
	3.配套视频，提供了本书各章所涉及的演示视频。
	4.readme.txt，本文件。

系统需求：

	其中html文件，可以使用浏览器直接打开。浏览器运行的系统需求如下。

    	1.Microsoft Internet Explorer 6.0或者更高版本。
                   2.HTML5+CSS3存在浏览器兼容问题，需要最新的谷歌浏览器，或者IE10
	3.Pentium III 或更高、500 Mhz（最低）、800 Mhz（建议）128M以上内存。
    	4.视频具有真彩色的 1024 x 768 VGA（最低），要求支持 Windows 的显示适配器。    
	5.至少300MB的硬盘存贮空间，64MB的交换空间。
	6.任意速度的光驱（仅用于安装）。

运行平台：
	
	Windows XP Professional、Windows XP Home、Windows Tablet PC、Windows 2003、Windows 7或更高版本Windows。

    	本光盘中的源程序以及资源文件仅可作为学习和参考之用，未经许可不得用于任何商业等其他用途。
