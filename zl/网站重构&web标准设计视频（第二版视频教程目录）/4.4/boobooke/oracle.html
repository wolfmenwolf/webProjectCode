  • 培训内容

    Oracle DBA核心篇 (培训学时：260学时)

    本部分的培训内容采用Oracle 9i几门长盛不衰的经典的课程作为培训教材。这些内容是成为Oracle DBA必须掌握的核心基础知识。这些课程包括：

    1.Oracle 9i DBA Fundamentals I

    2.Oracle 9i DBA Fundamentals II

    3.Oracle 9i DBA SQL

    4.Oracle 9i DBA Performance Tuning

    Unix/Linux管理基础培训篇(培训学时：50学时)

    Oracle和Unix紧密相连。掌握基本的Unix和Linux使用和管理技能是成为一名Oracle DBA必备的技能。本培训采用Linux和Sun Solaris两种常见的操作系统作为学习和练习的平台。培训内容包括如下部分：

    1.Unix/Linux基本入门，包含Linux和Solaris X86两种系统的讲解。

    2.Shell编程，包括Bash和Korn Shell

    3.Perl编程初步，如何使用Perl操纵Oracle数据库

    Oracle 专题研讨进阶篇(培训学时：50学时)

    目前Oracle 10g和11g越来越流行。本培训内容采用专题的形式，介绍10g/11g的新特性，以及Oracle DBA必须要掌握的技能：

    1.Oracle RAC(10g/11g)专题培训

    2.Oracle DataGuard专题培训

    3.Oracle ASM专题培训

    4.Oracle Enterprise Management Grid Control专题培训

    5. Oracle 10g/11g新特性培训
