MySQL 数据类型表
类型 大小 描述
字符串
CHAR(Length) Length字节 定长字段，长度0 ~ 255
VARCHAR(Length) String长度 + 1字节 变长字段，长度0 ~ 255
TINYTEXT String长度 + 1字节 字符串，最大长度为255
TEXT String长度 + 2字节 字符串，最大长度为65 535
MEDIUMTEXT String长度 + 3字节 字符串，最大长度为16 777 215
LONGTEXT String长度 + 4字节 字符串，最大长度为4 294 967 295
整型
TINYINT(Length) 1字节 范围：-128 ~ 127,或者0 ~ 255(无符号)
SMALLINT(Length) 2字节 范围：-32768 ~ 32767,或者0 ~ 65 535(无符号)
MEDIUMTINT(Length) 3字节 范围：-8 388 608 ~ 8388607,或者0 ~ 16 777 215(无符号)
INT(Length) 4字节 范围：-2 147 483 648 ~ 2 147 483 647,或者0 ~ 4 294 967 295(无符号)
BIGINT(Length) 8字节 范围：-9 223 372 036 854 775 808 ~ 9 223 372 036 854 775 807,
或者0 ~ 18 446 744 073 709 551 615(无符号)
浮点型
FLOAT 4字节 单精度浮点型，8位精度(4字节),m是十进制数字的总个数，d是小数点后面的数字个数
DOUBEL 8字节 双精度浮点型，16位精度(8字节)
定点数
DECIMAL Length + 1字节或Length + 2字节 存储为字符串DOUBEL，允许固定的小数
日期时间类型
DATE 3字节 采用YYYY-MM-DD格式
DATETIME 8字节 采用YYYY-MM-DD HH-MM-SS格式
TIMESTAMP 4字节 采用YYYYMMDDHHMMSS格式，可结束范围终止于2037
TIME 3字节 采用HH-MM-SS格式
枚举
ENUM 1或2字节 枚举，意味着每一列都可以具有多个可能的值之一
集合
SET 1，2，3，4或8字节 与ENUM一样，只不过每一列都可以具有多个可能的值
MySQL 数据类型属性
MySQL 关键字 含义
NULL 数据列可包含NULL值
NOT NULL 数据列不允许包含NULL值
DEFAULT 默认值，如果插入记录的时候没有指定值，将取这个默认值
PRIMARY KEY 主键
AUTO_INCREMENT 递增，如果插入记录的时候没有指定值，则在上一条记录的值上加1，仅适用于整数类型
UNSIGNED 无符号
CHARACTER SET name 指定一个字符集
