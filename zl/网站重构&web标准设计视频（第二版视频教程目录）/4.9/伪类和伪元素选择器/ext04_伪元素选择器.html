Mozilla Firefox (the official abbreviation for the Fx, an informal acronym for the FF), commonly known as Firefox (currently no official Chinese name), by the Mozilla Foundation and the open source group to develop the Web browser. Firefox from the Mozilla Application Suite derived Web browser, starting in 2005, every year the media has been selected as PC Magazine of the Year browser. According to statistics, as of May 2008 of the Firefox browser market share to 18.41 percent, second only to Microsoft's Internet Explorer.

The use of the Firefox open source web publishing engine Gecko, Gecko browser can allow as much as possible in accordance with the standard to show the contents of the page.

Firefox has a built-in tabbed browsing, spell check, instant bookmark, download manager, and custom search engines and so on. In addition, through third-party developers to contribute to the expansion of software to a wide range of functions, there are the more popular browser IE only page of IE Tab, to stop advertising pages of Adblock Plus, download movies online Video DownloadHelper, to protect computer security The NoScript, and so on. As of 2008, Firefox extensions can be used more than 2000 kinds of software.

Firefox can run multiple operating systems, the current stable version is on June 17, 2008 release of version 3.0, the source code to GPL / LGPL / MPL release of the three authorized methods.
